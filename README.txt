1. Run the java class "App" in the "Backend" folder to start the backend service;
2. Run "npm run start" in the "Frontend" folder to start the frontend;
3. After the above 2 steps completed successfully, to use this application, please visit "http://localhost:4200/" in the browser of the machine where this project is deployed or visit "http://{host}:4200".
4. You can search food facilities(trucks) by food, for e.g., you can enter "chicken" in the text box in the upper left corner of this webpage and press Enter or "Search" button, the search results will be displayed.
